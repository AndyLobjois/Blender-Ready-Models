# ⚠️ Model Replacement Process have an upgrade ! ⚠️

I've got several inconsistencies between character packages and it's not easy to maintain them in this state.
I decided to split the "raw" assets from the common assets shared between all packages.

1) **Tool package** (Bundle Build script, shaders, outline material, preset, version)
2) **Character packages** (FBX model, materials, textures, voices)
  

### Upgrades, Changes and Additions:

- Character animations packed inside the FBX (getting rid of the old animation way)
- Voices bundle
- Character menu poses fixed
- Windows and Linux bundle builds in the same time (check the updated [Model Replacement Tutorial](https://docs.google.com/document/d/1_EkomGUltoW7YjsjMeekDTd5ohrRellnOJj1eqVDPcI))
- TextureMod compatible
- Folders reordered by outfits
- Auto-delete .manifest files
- Attempt to fix all the known bugs
